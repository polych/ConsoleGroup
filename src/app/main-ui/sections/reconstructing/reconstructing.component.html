Реконструкція

ТОВ "Консоль БК" здійснює комплексне виконання робіт по реконструкції.

- Демонтажні роботи;
- Піскоструєння промислових залізобетонних площ;
- Всі види внутрішніх робіт, оздоблення інтер'єру;
- Гіпсокартонні роботи;
- Монтаж підлоги(плитка, паркет, паркетна дошка, ламінат, ковролін, вініл);
- Малярні роботи(штукатурення, шпаклювання та пофарбування стін та стель);
- Всі види оздоблення екстер'єру`;
